# MP3 Tag Reader and Editor (C)

## 📌 Description
This project is a C-based application that reads and modifies ID3 metadata tags in MP3 files. It uses binary file parsing and string manipulation to extract and update song information such as title, artist, and album.

## 🚀 Features
- Read MP3 metadata (ID3 tags)
- Edit title, artist, album fields
- Binary file parsing
- String manipulation for tag updates
- Error handling for invalid files

## 🛠️ Technologies Used
- C Programming
- File I/O (binary mode)
- String manipulation
- Structures and pointers
